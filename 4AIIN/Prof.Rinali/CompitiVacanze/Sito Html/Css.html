Uno dei punti negativi che abbiamo notato è che, per ogni tag che usiamo dobbiamo andare a cambiare i parametri.
Per "risolvere" questa nota negativa è stata creato il css.
Per css (Cascading Style Sheet, "Foglio di stile a cascata") si intende Un foglio di stile che ha al suo interno un insieme di regole che specificano l'aspetto degli elementi HTML in una pagina, cioè come gli elementi devono essere mostrati sullo schermo.

Per creare un folgio css,bisogna creare una cartella chiamata css e al suo interno creare un file nominato "main.css".



Poi per far si che la pagina vadà a prendere i valori indicati nel css, dobbiamo mettere un link è inserire i segiunti attributi.

per usare il css bisogna conoscere i selettori, che serveno per "identificare" i tag a cui cambiare i parametri. ecco una tabella in cui sono elencaati i principali.
